# vue-backstage
模仿高赞做的一个后台管理系统，主要使用vue2.0+vueCli4.0+Element ui
线上预览地址[backstage](http://120.77.176.152)
> 下载后npm i安装依赖，使用npm run serve运行开发环境
# 一丶主要页面
## 登录页->Login.vue
## 主页 Home.vue->Firs.vue
## 表单页->From.vue
## 404、403、500页面  ->404.vue、500.vue、403.vue
## 权限测试页 ->Permission.vue
## 高德地图页->ditu.vue
## 土狗播放器页->Music.vue
